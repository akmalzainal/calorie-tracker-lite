# Calorie Tracker Lite

A tiny **vanilla JavaScript** app to track daily calories.  
Add foods, see totals, set a daily limit with a live progress bar. Data is saved per-day using `localStorage`.

**Live Demo:** https://akmalzainal.github.io/calorie-tracker-lite/

## Features
- Add/delete entries
- Daily total + remaining calories
- Progress bar (turns red when over limit)
- Per-day persistence via `localStorage`
- “Reset Day” button

## Tech
- HTML, CSS, JavaScript (no frameworks)
